[ 前書き ]
このゲームは、世界中のどこでもダウンロードの必要がなくブラウザ上で遊ぶことができます。 さらにゲーム開始時に個人のアカウントを作成するので、いつでもゲームモード別の世界ランキングで自分の順位を確認することができます。 マルチプレイでは世界中のプレイヤーと対戦することができ、自分以外のプレイヤーがいない際はAIと対戦して遊ぶことが出来ます。 これからも日々改良していく予定です。ゲームモードも技も増やしていきたいと強く思っています。

趣味で毎日Unityをいじくりまわしている高校3年生です。毎日Unityを触っているので「毎日Unity」という名前でやっています。モデリングと作曲が少し苦手です。でもゲーム制作はとても好きなので、頑張って苦手を克服したいです。

( ゲームルールについて )

シングルプレイで遊ぶ時とマルチプレイで遊ぶ時とでやることが変わってくるので中止してください！


[ シングルプレイで遊ぶ時のルール ]
ゲームを開始すると47種類あるブロックの内、のどれかのブロックが決められた3色の内の1つの色がついた状態で連続して落下してきます。 これは決してランダムではなく、全プレイヤーを公平にするための法則を考え、適応させています。 落下してくるすべてのブロックは落下し終わると形が崩れます。 プレイヤーは、落下してくるブロックを操作し、同色のブロック同士をつなげて左右の壁を横断させると横断した同色のブロックは消去されスコアを稼ぐことができます。 ブロックを消した量によって得られるスコアは変動します。 さらにこのゲームには様々な難易度の技が存在し、ゲーム中にこれらの技を繰り出すことによってさらにボーナススコアを稼ぐことができます。 難しい技になればなるほど得られるスコアは多くなります。 ブロックを赤いライン以上積み上げてしまうとゲームオーバーになってしまい、そこでゲーム終了です。 シングルプレイで遊べる３つのゲームモードでは、これらのルールでスコアを稼ぎ、最高スコアを競い合って遊ぶことができます。


[ マルチプレイで遊ぶ時のルール ]
マルチプレイで遊ぶ際はシングルプレイで遊ぶ時のルールに追加ルールがあります。 マルチプレイでは、シングルプレイでのスコアの稼ぎ方でスコアではなくポイントを稼ぎます。このポイントは最高で15000ポイントまでしか稼ぐことができません。 プレイヤーは、稼いだポイントの内3000ポイントを消費して相手に攻撃することができます。 一度攻撃するのに必要なポイントは5000ポイントです。攻撃されたプレイヤーにはお邪魔ブロックが出現し、ゲームオーバーに近づきます。 攻撃するタイミングによっては攻撃されたプレイヤーが有利になる場合もあるので、少しタイミングを見極める必要があります。 マルチプレイで遊べる２つのゲームモードではこれらのルールで相手プレイヤーよりも長く生き残れるかを競い合って遊ぶことができます。 初めてこのゲームをプレイする人は、シングルプレイで練習してからマルチプレイで遊ぶことを勧めます。

( 法則性について )

[ ブロックとキューブの違い ]
まず、法則性を説明するにあたって「ブロック」と「キューブ」の違いを説明します。
プレイヤーが操作出来るのがブロックで、操作できないのキューブです。 ブロックはキューブの集まりであり、落下し終わると崩れてキューブになってしまいます。
下の画像を例に挙げると、赤いキューブの数が６個のブロックということです。

※このブロックは赤色ですが、ブロックの形と色は必ずしもこの画像通りだとは限りません。


[ ブロックの種類 ]
ゲームルールについてのところでも書きましたが、TheStackerOnlineでは、ゲームモードによっては最大で47種類のブロックが出現します。
下の画像がこのゲームに存在する全て(47種類)のブロックの一覧です。
一番キューブを含む数が少ないブロックのキューブの数は1個です。一番キューブを含む数が多いブロックのキューブの数は9個です。

※全てのブロックに色がついていますが、ブロックの形と色は、必ずしもこの画像通りだとは限りません。


[ 公平じゃないの！？ ]
上の画像にあるブロックは、赤、緑または青の色がついた状態で落下してきます。 そしてキューブは、同じ色同士でしか消去することができません。
だから「赤のブロックがすごい連続で落下してくるけど青のブロックが全く落ちてこない。」だったり「緑のブロックがばかり落ちてきて赤と青のブロックが少ししか落ちてこない。」などのことが起きてしまうと、プレイヤー間で公平ではなくなってしまいます。つまり有利なプレイヤーと不利なプレイヤーが出てきてしまうということです。
まとめると、赤、緑そして青のブロックがそれぞれ等しい割合で落下してこないとプレイヤー同士は公平ではなくなってしまうということです。


[ どうすれば公平になるの？ ]
上の画像内のすべてのブロックをキューブを含む数が少ない順でわかりやすく並べると下の画像のようになります。
色、形を考慮しないで、単純にキューブを含む数が異なるブロックを一個ずつ選んだとすると合計で9種類のブロックが選べると思います。
なぜここでキューブを含む数が異なる9種類のブロックを選んだかというと、キューブを含む数が同じブロックが連続で出現するのを避けるためです。
この9種類のブロックが含むキューブの数を全部をたすと、
1 + 2 + 3 + 4 + 5 + 6 + 7 + 8 + 9 = 45
になります。ブロックの色の種類は赤、緑、青の3種類だから3で割ると、
45 ÷ 3 = 15
になります。つまりキューブを含む数が異なる9種類のブロックが落下する毎に、落下した9種類のブロックが含むキューブの数が赤、緑、青でそれぞれ合計で15個になるようになれば良いということです。
これが実現できればキューブを含む数が同じブロックが連続で落下してくることもなく、キューブの色の数もそれぞれ等しくなるわけだからプレイヤー間で公平になります。

※全てのブロックに色がついていますが、ブロックの形と色は、必ずしもこの画像通りだとは限りません。


[ 例 ]
理解を深めてもらうためにここで一つ例を挙げたいと思います。
キューブを含む数が異なるブロックを9種類上の画像から選んで、下の画像のように色を付けたとします。
下の画像のように色がついていたとして赤、緑、青のキューブの数をそれぞれたしてみてください。
赤、緑、青、それぞれキューブの数の合計がそれぞれ15になりますよね？

ブロックが9個落下してくる都度このようになっていれば、プレイヤー間で公平になります。

( 技一覧 )

単純なゲームに見えて意外とたくさんの技があります。知っているだけでも有利になれると思うので、覚えておきましょう！ここに書いてある技以外にまだ存在するかもしれません...。


[ AllClear ]
フィールド上のキューブを全て消すたびに500のボーナススコアが得られます。一番簡単な技です。「気づいたらできてた！」なんてことも良くあります。



[ Combo ]
1つのブロックを設置することでキューブを連鎖的に消去するたびに1000のボーナススコア得られます。この技をどんどん繰り出せるようになると、マルチプレイで遊ぶ時に有利になれます。
※ボーナススコアが得られるのは、2Comboからです。



[ Ren ]
ブロックを設置するたびにキューブを消去すると、その都度300のボーナススコア得られます。
※ボーナススコアが得られるのは、2Renからです。



[ Double ]
2色のキューブを同時に消去するたびに2000のボーナススコア得られます。かなり難しい技です。



[ Triple ]
3色のキューブを同時に消去するたびに3000のボーナススコア得られます。難易度が高い分、得られるボーナススコアも多いです。

( ゲームモード一覧 )

リメイクということで、ゲームモードを5つに増やしました。まだまだ増やすよ!(つもり)


[ Normal ]
ブロックの種類は47種類で落下速度は普通です。10000スコア毎にブロックの落下速度が速くなります。このゲームモードが基本です。



[ HighSpeed ]
Normalのブロックの落下速度が速くなったバージョン。とても難易度が高く、初心者にはお勧めできません...。



[ Retro ]
Normalのブロックの種類が7種類になったバージョン。技も組みやすいので練習するならこのゲームモードがおすすめ！



[ VsAI ]
ブロックの種類は47種類でAIと対戦の練習ができるゲームモードです。VsPlayerモードの練習に活用してください。



[ VsPlayer ]
オンライン対戦モードです。VsAIモードの相手がプレイヤーになったバージョンです。はっきり言ってこのモードが一番楽しいです。
